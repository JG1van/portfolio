Penjelasan Syntax Bagian canvas

<img id="gambar" src="../assets/images/5zh1ey.jpg" alt="" width="220" height="277" /> - Elemen ini digunakan untuk menampilkan gambar di halaman web. Atribut src menentukan sumber gambar, alt memberikan deskripsi alternatif jika gambar tidak dapat ditampilkan, dan width serta height menetapkan ukuran gambar.

<canvas id="kanvas" width="220" height="277" style="border: 1px solid #d3d3d3"></canvas> - Elemen kanvas ini berfungsi sebagai area di mana gambar atau grafis dapat digambar menggunakan JavaScript. Atribut width dan height menentukan dimensi kanvas, sedangkan style memberikan gaya border pada kanvas.

<button onclick="kanvas()">Try</button> - Tombol ini ketika diklik akan memanggil fungsi kanvas(), yang bertugas untuk menggambar gambar pada kanvas.

function kanvas() { ... } - Fungsi ini digunakan untuk menggambar gambar pada kanvas. Di dalam fungsi, getContext("2d") digunakan untuk mendapatkan konteks 2D kanvas, dan drawImage(img, 0, 0, c.width, c.height) menggambar gambar pada posisi (0, 0) di kanvas dengan ukuran yang ditentukan.

<input type="file" id="fileInput" accept="image/*" /> - Elemen input ini memungkinkan pengguna untuk memilih file gambar dari perangkat mereka. Atribut type="file" menunjukkan bahwa elemen ini adalah untuk memilih file, dan accept="image/*" membatasi file yang dapat dipilih hanya pada gambar.

fileInput.addEventListener("change", (event) => { ... }); - JavaScript ini mendeteksi perubahan pada input file dan mengubah gambar yang ditampilkan di elemen <img>. Fungsi FileReader digunakan untuk membaca data gambar yang dipilih dan reader.onload mengubah sumber gambar agar sesuai dengan file yang baru saja dipilih.
