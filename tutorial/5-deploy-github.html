Tutorial Upload ke GitHub

Step 1: Buat Repository

Masuk ke akun GitHub Anda, klik "New Repository", masukkan nama repository, lalu klik "Create Repository".

Step 2: Buka Terminal di VS Code

Copy perintah yang diberikan di GitHub satu per satu ke terminal di VS Code untuk mulai menghubungkan project Anda ke repository.

Step 3: Inisialisasi Git

Masukkan perintah git init untuk menginisialisasi repository Git pada folder project Anda.

Step 4: Tambahkan File

Jalankan git add . untuk menambahkan semua file yang ingin Anda upload ke GitHub.

Step 5: Commit Perubahan

Gunakan perintah git commit -m "first commit" untuk menyimpan perubahan dengan pesan.

Step 6: Set Branch ke Main

Gunakan git branch -M main untuk mengatur branch utama menjadi "main".

Step 7: Tambahkan Repository Remote

Tambahkan repository remote Anda menggunakan git remote add origin https://github.com/JGsss/portofolio.git.

Step 8: Push ke Repository

Upload file Anda ke GitHub dengan perintah git push -u origin main.
